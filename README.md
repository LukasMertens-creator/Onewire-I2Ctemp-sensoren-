# Onewire-I2Ctemp-sensoren-
Opdracht 6 (per twee): lees twee Dallas Onewire temperatuursensoren uit en display deze in graden Celcius op het OLED display.  
Laat dezelfde schakeling ook nauwkeurig (met de 16 bits ADC) de uitgangsspanning meten van een spanningsdeler bestaande uit een serieschakeling van een 1 kOhm weerstand en een KTY81-220.   
Zorg ervoor dat deze waarde ook in graden Celcius op je OLED komt.
